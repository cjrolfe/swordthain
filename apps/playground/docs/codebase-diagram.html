3. Create company workflow (detail)

flowchart TB
    A["Issue opened\n(label: create-company)"]
    B["Checkout repo"]
    C["create_company.py"]
    D["Parse issue body:\nname, website, tone"]
    E["Copy company-template/\n→ <slug>/"]
    F["Fetch website\n(BeautifulSoup)"]
    G["OpenAI summary\n(optional)"]
    H["Playwright screenshot\n(optional)"]
    I["generate_sites.py"]
    J["Commit & push"]
    K["Comment & close issue"]

    A --> B --> C
    C --> D --> E
    C --> F
    F --> G --> H
    E --> I
    H --> I
    I --> J --> K

4. Archive / restore / delete workflow

flowchart TB
    A["Issue opened\n(Archive/Restore/Delete company: X)"]
    B["archive_company.py"]
    C{"Action?"}
    D["archive: set archived=true\nin sites.json"]
    E["restore: set archived=false\nin sites.json"]
    F["delete: remove from sites.json\n+ delete folder"]
    G["Commit & push"]
    H["Comment & close issue"]

    A --> B --> C
    C -->|archive| D
    C -->|restore| E
    C -->|delete| F
    D --> G
    E --> G
    F --> G
    G --> H
